Questions and Answers

Question What happens if the medical statement does not specify the medical condition, but only lists the foods to avoid?
Answer The medical statement can still be accepted, if it is signed by a licensed healthcare professional and includes a description (for example, Sam has a food allergy and can’t drink cow’s milk). It is appropriate to ask the participant if any other accommodations should be made to accommodate the medical condition.
Question How do we handle situations where “medical condition” is checked instead of “disability” on the current Medical Statement? Can we treat those as disabilities now and claim those meals?
Answer Yes, any current medical statements requiring meal modifications that are signed by a licensed healthcare professional will be accepted, whether “medical condition” or “disability” is checked. Meals can be claimed for those participants with signed medical statements.
Question Can you ask for clarification on which specific foods and beverages can be given, if the medical statement is not clear?
Answer Yes, it is reasonable to ask for more information about specific foods and beverages to be given to a participant, in order to accommodate the medical condition.
Question Can we accept a doctor’s note on letterhead or a prescription pad instead of the state agency’s form?
Answer Yes, a doctor’s note on letterhead or a prescription pad is acceptable documentation if the note includes the doctor’s signature and the medical reason for the meal substitution.
